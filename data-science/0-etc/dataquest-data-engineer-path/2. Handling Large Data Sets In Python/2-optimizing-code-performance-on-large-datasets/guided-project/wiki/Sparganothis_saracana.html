Sparganothis saracana is a species of moth of the Tortricidae family. It is found in the United States, including Alabama, Arkansas, Florida, Louisiana, Maryland, Mississippi, Missouri, New Jersey, North Carolina, Tennessee, Texas, Virginia and West Virginia.[2]

The wingspan is 17–22 mm.
