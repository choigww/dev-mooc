Code page 1023

From Wikipedia, the free encyclopedia
Jump to: navigation, search

Code page 1023, also known as CP1023[1] or E7DEC,[2] is an IBM code page number assigned to the Spanish variant of DEC's National Replacement Character Set (NRCS).[3][4] The 7-bit character set was introduced for DEC's computer terminal systems, starting with the VT200 series in 1983, but is also used by IBM for their DEC emulation. Similar but not identical to the series of ISO 646 character sets, the character set is a close derivation from ASCII with only eight code points differing.
