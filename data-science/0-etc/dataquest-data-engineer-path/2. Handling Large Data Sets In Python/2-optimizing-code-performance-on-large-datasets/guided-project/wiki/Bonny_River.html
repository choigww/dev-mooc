Bonny River

From Wikipedia, the free encyclopedia
Jump to: navigation, search

Bonny River is a river in Rivers State, Nigeria.[1] Water taxis which move along the river provide the connection between Bonny Island and Port Harcourt,[1] the capital of Rivers State, which lies alongside the river.[2]
