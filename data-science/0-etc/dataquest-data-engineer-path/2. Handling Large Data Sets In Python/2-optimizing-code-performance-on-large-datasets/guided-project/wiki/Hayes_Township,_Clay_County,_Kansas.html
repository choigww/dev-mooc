Hayes Township is a township in Clay County, Kansas, USA. As of the 2000 census, its population was 206.

Geography[edit]

Hayes Township covers an area of 35.67 square miles (92.4 km2) and contains no incorporated settlements. According to the USGS, it contains four cemeteries: Brethren in Christ, Greenwood, Hayes and Pleasant Hill.

Transportation[edit]

Hayes Township contains one airport or landing strip, Holmes Airpark.
