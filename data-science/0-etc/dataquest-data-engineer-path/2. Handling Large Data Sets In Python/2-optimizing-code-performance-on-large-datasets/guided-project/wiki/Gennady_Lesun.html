Gennady Nikolayevich Lesun (Russian: Геннадий Николаевич Лесун; born 5 September 1966) is a retired Belarusian professional footballer and currently a coach.

Honours[edit]

Dinamo Minsk

  • Belarusian Premier League champion: 1992, 1992–93, 1993–94
  • Belarusian Cup winner: 1992
