Swathi Chinukulu is a 1989 Telugu-language Indian feature film directed by Sri Chakravarthy for producer T.Prasad, starring Vanisri, Ramya Krishnan, Sharat Babu, Suresh and Jayasudha in lead roles.[1][2]
