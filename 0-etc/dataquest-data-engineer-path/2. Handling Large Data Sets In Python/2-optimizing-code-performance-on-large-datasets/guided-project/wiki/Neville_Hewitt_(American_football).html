Neville Hewitt (born April 6, 1993) is an American football inside linebacker for the Miami Dolphins of the National Football League (NFL). He played college football at Marshall University, where he was a safety and linebacker. Hewitt signed with the Miami Dolphins as an undrafted free agent in 2015.

Professional career[edit]

On May 2, 2015, Hewitt signed with the Miami Dolphins as an undrafted free agent following the conclusion of the 2015 NFL Draft.[1]
