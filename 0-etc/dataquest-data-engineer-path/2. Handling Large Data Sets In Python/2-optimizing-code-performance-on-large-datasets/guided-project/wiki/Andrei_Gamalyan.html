Andrei Sarkisovich Gamalyan (Russian: Андрей Саркисович Гамалян; born 27 January 1996) is a Russian football player of Armenian descent.

He made his debut in the Russian Professional Football League for FC Krasnodar-2 on 13 October 2013, in a game against FC Volgar Astrakhan.[1]

On 19 June 2014, Gamalyan joined Armenian Premier League side FC Shirak on a six-month loan.[2]
