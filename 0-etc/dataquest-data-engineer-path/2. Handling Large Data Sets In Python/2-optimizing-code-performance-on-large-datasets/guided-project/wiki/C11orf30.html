Protein EMSY is a protein that in humans is encoded by the C11orf30 gene.[3]

Clinical significance[edit]

C11orf30 has been shown to associate with atopy and susceptibility to poly-sensitisation.[4]

Interactions[edit]

C11orf30 has been shown to interact with ZMYND11,[5] BRCA2[5] and CBX1.[5]
