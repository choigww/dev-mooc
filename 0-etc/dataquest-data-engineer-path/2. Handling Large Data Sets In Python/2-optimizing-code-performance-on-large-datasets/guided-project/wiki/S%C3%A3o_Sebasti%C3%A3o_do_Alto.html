Coordinates: 21°57′25″S 42°08′06″W﻿ / ﻿21.95694°S 42.13500°W﻿ / -21.95694; -42.13500 São Sebastião do Alto is a municipality located in the Brazilian state of Rio de Janeiro. Its population was 8,747 (2005) and its area is 397 km².[1]
