Fischli

Fischli is a surname. Notable people with the surname include:

  • Paul Fischli (born 1945), Swiss footballer and manager
  • Peter Fischli (born 1952), Swiss artist
This page lists people with the surname Fischli. If an internal link intending to refer to a specific person led you to this page, you may wish to change that link by adding the person's given name(s) to the link.
