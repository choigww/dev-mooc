Berzeliustinden

From Wikipedia, the free encyclopedia
Jump to: navigation, search

Berzeliustinden is a mountain in Wedel Jarlsberg Land in Spitsbergen, Svalbard. It has a height of 1,211 m.a.s.l.[1] It is located south of Van Keulenfjorden, and is included in the Sør-Spitsbergen National Park.[2]

The mountain is named after Swedish chemist Jöns Jacob Berzelius.[1]
