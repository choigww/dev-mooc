Ordinary is an unincorporated community in Gloucester County, Virginia, United States. Ordinary is located along U.S. Route 17 4.5 miles (7.2 km) north of Gloucester Point. Ordinary has a post office with ZIP code 23131.[2]

The community was named after the local ordinary (i.e. public dining hall).[3] Woodville School was added to the National Register of Historic Places in 2004.[4]
