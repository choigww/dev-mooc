Kul Gul (Persian: كول گل‎‎, also Romanized as Kūl Gūl)[1] is a village in Tang-e Haft Rural District, Papi District, Khorramabad County, Lorestan Province, Iran. At the 2006 census, its existence was noted, but its population was not reported.[2]
