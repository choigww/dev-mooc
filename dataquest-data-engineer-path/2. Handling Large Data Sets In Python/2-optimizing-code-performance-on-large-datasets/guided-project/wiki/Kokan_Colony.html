Kokan Colony

From Wikipedia, the free encyclopedia
Jump to: navigation, search

Kokan Colony is one of the neighbourhoods of Baldia Town in Karachi, Sindh, Pakistan.[1]

There are several ethnic groups in neighbourhood including Muhajirs, Sindhis, Kashmiris, Seraikis, Pakhtuns, Balochis, Memons, Bohras, Ismailis, etc. Over 99% of the population is Muslim. The population of Baldia Town is estimated to be nearly one million.
