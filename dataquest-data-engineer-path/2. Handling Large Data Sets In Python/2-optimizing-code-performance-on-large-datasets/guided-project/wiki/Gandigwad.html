Gandigwad is a village in the southern state of Karnataka, India.[1][2] It is located in the Khanapur taluk of Belgaum district in Karnataka.

Demographics[edit]

As of 2001[update] India census, Gandigwad had a population of 6019 with 3120 males and 2899 females.[1]
