Bye! My Boy! is The Brilliant Green's ninth single, released in 1999. It peaked at #29 on the Oricon singles chart.[1][2][3][4]

Track listing[edit]

All lyrics written by Tomoko Kawase; all music composed by Shunsaku Okuda.
