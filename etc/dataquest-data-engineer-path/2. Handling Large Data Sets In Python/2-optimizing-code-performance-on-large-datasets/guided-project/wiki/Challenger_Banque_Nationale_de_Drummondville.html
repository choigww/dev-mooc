The Challenger Banque Nationale de Drummondville is a professional tennis tournament held in Drummondville, Quebec, Canada since 2015.[1] It was held in Rimouski, Quebec, Canada from 2006 to 2014. The event is part of the ATP Challenger Tour and is played on indoor hard courts.
