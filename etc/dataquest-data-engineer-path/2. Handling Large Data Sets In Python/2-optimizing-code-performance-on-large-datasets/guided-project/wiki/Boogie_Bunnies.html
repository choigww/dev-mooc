Boogie Bunnies
Developer(s) Artech Studios
Publisher(s) Sierra Online
Platform(s) Xbox 360 (XBLA), PC
Release date(s) January 16, 2008
Genre(s) Puzzle game
Mode(s) Single player, Multiplayer (co-op)

Boogie Bunnies is a match-3 puzzle game developed by Artech Studios for the Xbox 360's Xbox Live Arcade service and Windows PC's. The game features same machine and Xbox Live co-op multiplayer. The title was released on January 16, 2008.[1]

Reception[edit]

Boogie Bunnies gameplay screenshot

Team Xbox scored the game a 7.5 out of 10 stating, "Despite a few bungled dance steps here and there, Boogie Bunnies is still a strangely addictive experience."[2] IGN criticized the game saying "It does nothing different from the gaggle of puzzlers that are already out there and the old playstyles that it employs aren't nearly as fun as we'd hoped."[3]

Boogie Bunnies was nominated for one Xbox Live Arcade 2008 award, "Best Family Game".[4]
