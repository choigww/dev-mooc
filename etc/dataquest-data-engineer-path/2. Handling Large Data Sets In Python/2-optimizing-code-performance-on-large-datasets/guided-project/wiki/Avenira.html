Avenira

From Wikipedia, the free encyclopedia
Jump to: navigation, search

Avenira (Russian: Авени́ра) is a Russian non-canonical female first name.[1] Its masculine version is Avenir.[1]

The diminutives of "Avenira" are Avenirka (Авени́рка), Ava (А́ва), Venya (Ве́ня), Vena (Ве́на), Vira (Ви́ра), and Nira (Ни́ра).[1]
